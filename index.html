Make sure you're already logged into your school's Canvas in a browser tab before you start.
1
In Canvas, click your profile picture or name in the top-left corner of the screen.
2
Choose Settings from the menu that appears.
3
Scroll all the way down to the Approved Integrations section near the bottom of the page.
4
Click + New Access Token. For the purpose, type anything you like — "StudyFlow" works great.
5
Click Generate Token, then copy the long token shown and paste it into the Canvas API Token field above. Canvas only shows it once, so copy it right away!
